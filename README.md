Para compilar e gerar o executável:
Dentro do diretório raíz no seu terminal rode: <br>
    ' gcc ./src/main.c -o RedeNeuralBP -lm '

Para rodar o programa: ./RedeNeuralBP treinamento.csv teste.csv
Ou apenas ./RedeNeuralBP
